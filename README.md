## Usage

In the project directory, you can run:

```sh
yarn install
```

Run the application:

```sh
yarn start
```

Run all tests:

```sh
yarn test
```

Open [localhost:3000](http://localhost:3000) or [192.168.1.9:3000](http://192.168.1.9:3000) to view it in the browser.
